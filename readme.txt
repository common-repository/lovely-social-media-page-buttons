== Description ==

Lovely Social Media Page Buttons plugin let you add animated social media page icons to the sidebar using the widget or inside page/post using the shortcode [lovelysocialpagebuttons].

Check out the <a href="http://techsini.com/our-wordpress-plugins/lovely-social-media-page-buttons" target="_blank">Lovely Social Media Page Buttons plugin demo here</a>.
